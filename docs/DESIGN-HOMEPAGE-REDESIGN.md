## 模块样式规范

### 1. 鱼情指数卡片
- **背景**：渐变色 `#1A3A5C → #2D5A87`
- **大数字**：白色，48px，加粗
- **评分条**：白色半透明背景，进度条白色
- **推荐文字**：白色 70% 透明度，14px

### 2. 排行榜 Tab
- **标签样式**：底部边框高亮当前选中
- **选中态**：`color: #1A3A5C; font-weight: 600`
- **默认态**：`color: #666`

### 3. TOP3 卡片样式
| 名次 | 背景色 | 边框/图标 |
|------|--------|----------|
| 🥇 第1名 | 金色渐变 `#FFD700 → #FFA500` | 皇冠图标 |
| 🥈 第2名 | 银色渐变 `#C0C0C0 → #A0A0A0` | 银色图标 |
| 🥉 第3名 | 铜色渐变 `#CD7F32 → #B87333` | 铜色图标 |
- **卡片**：白色背景，圆角 12px，轻微阴影
- **字体**：第1名 16px 加粗，其他 14px

### 4. 鱼种排行
- **布局**：2列网格
- **名次**：左侧图标（第1名金色，第2名银色，第3名铜色）
- **数字**：`color: #1A3A5C; font-weight: 600`

### 5. 附近钓点列表
- **卡片**：白色背景，左侧地图图标，右侧距离+评分
- **距离**：蓝色小字 `color: #0D7377`
- **评分**：星星图标 + 数字
- **分割线**：底部 1px `#f0f0f0`

## 颜色规范

| 用途 | 颜色 | Hex |
|------|------|-----|
| 主色 | 深蓝 | `#1A3A5C` |
| 强调色 | 青绿 | `#0D7377` |
| 背景色 | 浅灰 | `#F5F7FA` |
| 文字主色 | 深灰 | `#1A2634` |
| 文字次色 | 中灰 | `#666666` |
| 第1名 | 金色 | `#FFD700` |
| 第2名 | 银色 | `#C0C0C0` |
| 第3名 | 铜色 | `#CD7F32` |

## 字体规范

| 用途 | 字号 | 字重 | 颜色 |
|------|------|------|------|
| 页面标题 | 18px | 600 | `#1A2634` |
| 模块标题 | 15px | 600 | `#1A2634` |
| 正文 | 14px | 400 | `#374151` |
| 辅助文字 | 12px | 400 | `#9CA3AF` |
| 数字强调 | 16px | 700 | `#1A3A5C` |

## 间距规范

- **页面边距**：16px
- **模块间距**：12px
- **卡片内边距**：14px
- **圆角**：12px（卡片）、8px（按钮/输入框）

## MVP 实现优先级

1. **P0（必须）**：鱼情指数卡片 + 排行榜Tab切换 + TOP3展示
2. **P1（重要）**：鱼种排行 + 附近钓点列表
3. **P2（迭代）**：地图交互、详细筛选条件
