# 钓迹 App 首页重构 - 设计方案（v2）

## 核心变更

> 2026-04-20 更新：移除「鱼情指数」模块，替换为「钓点热度榜」

**变更原因：** 鱼情指数依赖的历史数据（上钩率、用户评分）在 MVP 阶段数据量不足，无法提供可信结果。改为统计真实上报次数，更直观、数据基础更扎实。

## 模块样式规范

### 1. 钓点热度榜卡片
- **背景**：渐变色 `#1A3A5C → #2D5A87`
- **卡片内背景**：白色 10% 透明度，柔和玻璃质感
- **时间 Tab**：胶囊样式，当前选中白色背景

### 2. TOP3 样式（热度榜 + 用户榜共用）

| 名次 | 背景 | 文字颜色 | 图标 |
|------|------|---------|------|
| 🥇 第1名 | 金色渐变 `#FFD700 → #FFA500` | `#5D3A00` 深金 | 皇冠 + 数字 |
| 🥈 第2名 | 银色渐变 `#D0D0D0 → #A8A8A8` | `#333333` 深灰 | 数字 |
| 🥉 第3名 | 铜色渐变 `#CD7F32 → #B87333` | `#4A2000` 深棕 | 数字 |
| 4名后 | 白色背景 | 默认色 | 无图标 |

- **数字文字**：28px 加粗
- **标签文字**（次/本月）：12px，白色半透明

### 3. 用户排行榜卡片
- **卡片**：白色背景，圆角 12px，轻微阴影
- **头像**：40px 圆形
- **昵称**：14px 加粗
- **数值**：16px，`color: #1A3A5C`

### 4. 鱼种排行
- **布局**：2列网格
- **名次**：左侧（第1名金色，第2名银色，第3名铜色）
- **数量**：`color: #1A3A5C; font-weight: 600`

### 5. 附近钓点列表
- **卡片**：白色背景，左侧地图图标，右侧距离+评分
- **距离**：蓝色小字 `color: #0D7377`
- **评分**：星星图标 + 数字
- **分割线**：底部 1px `#f0f0f0`

## MVP 实现范围

| 模块 | 优先级 | MVP 内容 |
|------|--------|---------|
| 钓点热度榜（顶部卡片） | P0 | 近30天 / 本月 切换，TOP3 金银铜样式 |
| 用户排行榜 Tab | P0 | 全国榜 + 省级榜，前三名突出样式 |
| 鱼种排行 | P1 | 默认展示综合热门鱼种，TOP10 |
| 附近钓点列表 | P1 | 列表视图，距离升序 |

## 交互说明

- **热度榜 Tab 切换**：点击「近30天」或「本月」，无刷新加载
- **热度榜 → 排行榜**：点击「查看完整排行」跳转排行榜页
- **附近钓点 → 钓点详情**：点击卡片跳转钓点详情页
- **列表下拉刷新**：支持下拉刷新最新数据
